## Contenidos

* Plan de Pruebas(PP) para el proyecto **Hoptolt**. Este plan de pruebas define los objetivos, alcance y casos de prueba para evaluar las funcionalidades definidas en el SRS, y las historias de usuario(HU).
* Especificación de Casos de Prueba(ECP) para el proyecto **Hoptolt**. Este documento detalla de manera individual la ejecución y los resultados obtenidos para los casos de prueba definidos en el PP.
